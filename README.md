## Quick Start

```bash
# Clone the repository
git clone https://github.com/efww/rpg_game.git
cd rpg_game

# Run the game
cargo run --bin rpg_seamless
```

## Controls

- **WASD** - Move character
- **Space** - Attack
- **R** - Restart (when game over)
